The Negotiator — Study Abroad Edition

Voice agents that phone UK study-abroad consultancies for a Pakistani student, extract itemised quotes, negotiate with database-sourced leverage, and produce a ranked GBP + PKR report. Every number the agent says on a call is traceable to a tool response.

Honesty guardrails — enforced by architecture, not prompts

Leverage cannot be invented

Competitor figures exist for the agent only inside a get_leverage response, and the SQL behind it only surfaces calls that ended as complete quotes. No tool response — the figure does not exist.

Benchmarks stay server-side

The agent never does fee arithmetic. red_flag_check compares against published deposits and the market floor (source URLs in config) and returns the exact sentence to say.

Profile is frozen, verbatim

Grades, IELTS, and funds are injected from the frozen profile only. Anything not in it is answered with “I'd have to check” — estimating is forbidden by the prompt's honesty block.

AI disclosure on request

Asked “are you an AI?”, the agent discloses honestly with a fixed line, then returns to its last unanswered question. Never denies, never volunteers.
